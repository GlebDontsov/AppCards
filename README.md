# AppCards

Приложение для узучения английского языка.

## Возможности
- создание своего набора карточек

![Снимок экрана от 2023-01-26 12-43-28](https://user-images.githubusercontent.com/103101200/214804983-90e3e4d7-5921-4060-b78e-e06dafeac583.png)

- удаление набора карточек
- изменение существуюших наборов карточек
- тренировка
- имеется звуковое сопровождение карточек
- создано 52 набора на разные темы
